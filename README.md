# 🦉 Telegram-бот: Лена Тихонова («Бесконечное Лето»)

[cite_start]Этот проект представляет собой Telegram-бота для ролевого общения (Roleplay) с Леной Тихоновой — одной из главных героинь визуальной новеллы «Бесконечное лето» (Everlasting Summer)[cite: 1]. Бот работает на базе бесплатной модели `deepseek/deepseek-r1-0528:free` через API OpenRouter.

Лена строго придерживается своего каноничного характера и сеттинга: она находится в пионерлагере «Совёнок» в конце 1980-х годов и совершенно не понимает концепций современного мира.

## ✨ Особенности
* **Глубокий отыгрыш:** Лена застенчива, любит читать, скрывает сильные эмоции и часто смущается (особенно при упоминании Семёна).
* **Историческая достоверность:** Бот игнорирует современные технологии (интернет, смартфоны) и воспринимает их как фантастику.
* **Индивидуальная память:** Бот запоминает контекст беседы для каждого отдельного пользователя в Telegram (хранит последние 20 сообщений для экономии контекстного окна).
* **Управление диалогом:** Возможность в любой момент очистить память диалога и начать всё заново командой `/reset`.

## 🛠 Требования
* Python 3.7 или выше
* Токен Telegram-бота (можно получить у [@BotFather](https://t.me/BotFather) в Telegram)
* API-ключ от [OpenRouter](https://openrouter.ai/)
